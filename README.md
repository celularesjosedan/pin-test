# 🔐 Control de PINES

Herramienta interactiva para desbloqueo de celulares, con lista de PINs más comunes y probabilidad de éxito.  
Diseño **oscuro con acentos neón** (sin animaciones, máxima velocidad).

## 🚀 Características

- **50 PINs de 4 dígitos y 50 de 6 dígitos** más usados mundialmente.
- **Probabilidad de éxito** calculada por ranking de frecuencia.
- **Marcado visual** de PINs probados (tachado + check verde neón).
- **Estadísticas en tiempo real** (pendientes, probados, total).
- **Persistencia local** (guardado automático en el navegador).
- **Diseño oscuro con neón** (fondo #0b0f1a, acentos #00d4ff, #00f0c0, #ff7b9c).
- **Sin animaciones ni transiciones pesadas**, solo cambios de color al hover.

## 📦 Uso

1. Abre `index.html` en cualquier navegador.
2. Haz clic en un PIN para marcarlo como probado (o desmarcarlo).
3. Los datos se guardan solos.
4. Usa el botón **Reiniciar todo** para resetear.

## 📊 Probabilidad

Basada en la frecuencia relativa de cada PIN en estudios reales (DataGenetics, filtraciones).  
Los primeros puestos tienen mayor porcentaje.

## 🛠 Tecnologías

- HTML5, CSS3, JavaScript vanilla
- localStorage

- `index.html` — todo en uno.
